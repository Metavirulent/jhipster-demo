Technology Stack

Here's an incomplete list of technologies used in the demo.

Spring Boot

On the server side, the demo app is based on Spring Boot which provides an oppinionated view on setting up Spring based applications. This reduces boilerplate code and avoids tedious XML configuration while providing a production ready framework. Behind the scenes, it uses popular Spring technologies like Web, Security, Mail, JPA, and others.

Learn more »

AngularJS

The front-end is built using AngularJS. The single-page web technology allows snappy clients that almost feel like a rich client. It also contains other cool frameworks like Twitter Bootstrap and HTML5 Boilerplate as well as the SASS CSS extension. The resulting client is fully responsive and mobile ready. The client uses REST to talk to the server.

Learn more »

JHipster

JHipster was used for scaffolding the Demo App. It combines AngularJS and Spring Boot into a comprehensive full stack using best practices for these technologies. It is based on a Yeoman generator and can be used to keep the client-side and server-side model in sync. It also combines Grunt and Maven for a seamless build cycle.
